description: Scheduling with Massively Heterogeneous Resource Requirements
abstract: |
    In modern computing systems, jobs resource requirements are massively heterogenous by default. Schedulers must allocate CPU, GPU, memory, and network resources, to name but a few. While some systems, such as cloud VM schedulers, allocate a handful of standardized resource bundles, many systems are flexible to exact user requirements, coping with the resulting heterogeneity. Existing queueing-theoretic performance analysis cannot handle this heterogeneity.

    I will present a family of scheduling policies designed to cope with practical resource heterogeneity, with proven optimal throughput and low simulated latency on datacenter trace data.
